<h4>Usage</h4>

    $ git clone https://github.com/akuhidayat/reeloader.git
    $ cd reeloader

    > Run With NodeJs
      $ node index.js
      * Filename with extentions (ex: video.mp4)

    > Run With Python
      ! Optional Command
        $ pip install colorama rquests
                 or
        $ python3 -m pip install colorama requests

      $ python3 insta.py
      * Filename without extentions (ex: video)
